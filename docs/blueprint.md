# **App Name**: Atelier Luxe

## Core Features:

- Hero Section: Elegant hero section with designer bio, brand story, and prominent "Start Designing" CTA.
- Portfolio Gallery: Masonry-style gallery with high-resolution images of luxury bag designs. Modal pop-ups for detailed views and design info.
- Base Silhouette Selection: Allows users to choose from a set of base bag silhouettes as a foundation for their design.
- Texture & Material Configurator: Configurator enabling users to apply textures/materials, masked to the selected bag shape. The configurator uses layering with CSS absolute positioning, masking (mask-image or mix-blend-mode), and z-index for layering order.
- Hardware Customization: Selection tool for hardware options like buckles and buttons with changeable colors.
- Stitching Options: Selection tool offering diverse stitching styles and color choices to customize the bag's detailing.
- Design Storage: Store user-generated bag designs using Firestore, including references to selected layer IDs for later retrieval and sharing.

## Style Guidelines:

- Primary color: Black (#000000), evoking luxury and sophistication.
- Background color: White (#FFFFFF), providing a soft, neutral backdrop.
- Accent color: Light Grey (#D3D3D3) for hardware and highlights.
- Headline font: 'Playfair', a modern serif with an elegant, fashionable feel.
- Body font: 'PT Sans', a humanist sans-serif providing a modern, readable counterpoint to 'Playfair'.
- Use minimalist, custom-designed icons to represent bag shapes, materials, and hardware options.
- Clean, spacious layout with a focus on showcasing high-resolution images and the configurator tool.
- Subtle transitions and animations to enhance user experience and provide visual feedback during configuration.
